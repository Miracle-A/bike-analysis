Miracle Bike Data Analysis Project

Project Overview

This project focuses on analyzing Miracle Bike's key performance metrics, such as hourly revenue, profit trends, seasonal patterns, and rider demographics, using SQL and Power BI.

Project Highlights

  • Created an SQL database and imported data from Excel files.
  • Developed SQL queries to extract relevant insights.
  • Designed an interactive Power BI dashboard.
  • Provided recommendations for future pricing strategies.

Project Files

  • Bike Share Year 0 Data
  • Bike Share Year 1 Data
  • Cost Table Data
  • Download Power BI Dashboard
  • SQL Script

Central Analysis Question

What pricing strategy should we implement next year to maximize profitability without reducing demand?

Recommendations Summary

Based on our analysis, we recommend a **10-15% price increase** to balance growth and profitability. With a previous 25% increase, the demand increased by 64%, indicating that riders are likely to tolerate another moderate increase.
